# yemek-siparis-uygulamasi

## Geliyo - Masaüstü Sipariş Otomasyonu

"Geliyo", kullanıcıların ürün seçimi yapabildiği ve sepet tutarını hesaplayabildiği, C# tabanlı bir masaüstü sipariş uygulamasıdır. 

## Özellikler 
* **Kullanıcı Arayüzü (UI):** Siparişlerin kolayca seçilip yönetilebildiği masaüstü arayüz.
* **Sepet Yönetimi:** Eklenen ürünlerin listelenmesi ve toplam tutarın dinamik olarak hesaplanması.
* **Veri Akışı:** Kullanıcı seçimlerinin uygulama içinde tutulması ve işlenmesi.

## Kullanılan Teknolojiler
* **Programlama Dili:** C#
* **Framework:** .NET Framework (Windows Forms / WPF)
* **Geliştirme Ortamı:** Visual Studio
